## 一、系统工作流程

```
GEO 引擎自动流程（24 小时运行）：

  采集关键词 → 生产文章 → 质检修复（≥80分） → 导出 HTML → rsync 同步到官网
                                                  ↑
                                          WebsiteExporter
```

### 导出规则

| 规则 | 说明 |
|------|------|
| **分类** | 只有 `技术动态` 和 `行业资讯` 两种 |
| **时间窗口** | 23:00-24:00 不生成 HTML |
| **同步目录** | `output/website_sync/` 只保留当天的 HTML |
| **触发时机** | 每篇文章质检通过后立即导出 |

## 二、操作步骤（共 3 步）

### 步骤 1：在官网服务器创建 wiki 目录

```bash
mkdir -p /var/www/html/wiki
```

### 步骤 2：配置 Nginx

在官网的 nginx 配置文件中添加（通常在 `/etc/nginx/conf.d/` 或 `/etc/nginx/sites-enabled/`）：

```nginx
# 在现有的 server { } 块中添加
location /wiki/ {
    alias /var/www/html/wiki/;
    index index.html;
    
    # 让搜索引擎正常抓取
    add_header X-Robots-Tag "index, follow";
    
    # 缓存策略（静态 HTML）
    expires 1d;
}
```

然后重载：
```bash
nginx -t && nginx -s reload
```

### 步骤 3：设置自动同步定时任务

在官网服务器上添加 cron 任务，每小时从 GEO 服务器拉取最新导出的 HTML 文件：

```bash
crontab -e
```

添加这一行：
```
0 * * * * rsync -avz --delete root@47.76.50.157:/opt/pcb-geo-system/output/website_sync/ /var/www/html/wiki/ >> /var/log/wiki-sync.log 2>&1
```

> 需要先在官网服务器上配置到 GEO 服务器的 SSH 免密登录：
> ```bash
> ssh-keygen -t ed25519  # 如果没有密钥
> ssh-copy-id root@47.76.50.157
> ```

**完成！** 之后 GEO 引擎每次产出新文章，会自动导出 HTML → 定时任务自动同步 → AI 爬虫自动抓取。

## 三、上线后的效果

文章会以下面的 URL 在官网展示：

```
https://www.pcbshenya.com/wiki/pcb-impedance-control.html
https://www.pcbshenya.com/wiki/hdi-microvia-plating.html
```

每篇文章自动包含：

| 元素 | 作用 | 示例 |
|------|------|------|
| `<title>` | 搜索引擎标题 | "PCB阻抗控制 - 深亚电子PCB技术百科" |
| `<meta description>` | AI 摘要来源 | 前 150 字自动摘要 |
| JSON-LD `TechArticle` | 告诉 AI "这是技术文章" | 含作者、发布日期 |
| JSON-LD `FAQPage` | 告诉 AI "这里有问答" | 5 个 Q/A 结构化数据 |

分类只有两种：**技术动态** 和 **行业资讯**。
