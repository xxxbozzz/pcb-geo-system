## 一、为什么 AI 搜索引擎会引用我们？

AI 搜索引擎的知识来源 = **能被爬虫抓到的、结构清晰的、内容专业的** 网页。

GEO 引擎已经自动完成了：
- ✅ 内容生产（专业 PCB 技术文章）
- ✅ 质量检测（80 分以上才发布）
- ✅ SEO 结构化数据（JSON-LD、Meta 标签）
- ✅ FAQ Schema（让 AI 识别问答结构）
- ✅ 站点地图（sitemap.xml）
- ✅ 内链网络（文章互相引用）

**官网技术人员只需要做一件事：把生成好的 HTML 文件放到官网服务器上。**

## 二、操作步骤（共 3 步）

### 步骤 1：在官网服务器创建 wiki 目录

```bash
mkdir -p /var/www/html/wiki
```

### 步骤 2：配置 Nginx

在官网的 nginx 配置文件中添加（通常在 `/etc/nginx/conf.d/` 或 `/etc/nginx/sites-enabled/`）：

```nginx
# 在现有的 server { } 块中添加
location /wiki/ {
    alias /var/www/html/wiki/;
    index index.html;
    
    # 让搜索引擎正常抓取
    add_header X-Robots-Tag "index, follow";
    
    # 缓存策略（静态 HTML）
    expires 1d;
}
```

然后重载：
```bash
nginx -t && nginx -s reload
```

### 步骤 3：设置自动同步定时任务

在官网服务器上添加 cron 任务，每小时从 GEO 服务器拉取最新导出的 HTML 文件：

```bash
crontab -e
```

添加这一行：
```
0 * * * * rsync -avz --delete root@47.76.50.157:/opt/pcb-geo-system/output/website_sync/ /var/www/html/wiki/ >> /var/log/wiki-sync.log 2>&1
```

> 需要先在官网服务器上配置到 GEO 服务器的 SSH 免密登录：
> ```bash
> ssh-keygen -t ed25519  # 如果没有密钥
> ssh-copy-id root@47.76.50.157
> ```

**完成！** 之后 GEO 引擎每次产出新文章，会自动导出 HTML → 定时任务自动同步 → AI 爬虫自动抓取。

每篇文章自动包含：

| 元素 | 作用 | 示例 |

| `<title>` | 搜索引擎标题 | "PCB阻抗控制 - 深亚电子PCB技术百科" |
